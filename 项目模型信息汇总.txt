1. Stable Diffusion模型 (sd_model.py)
================================================================================

【模型功能】
- 核心功能：文本生成图像（文生图）
- 支持功能：根据文本描述生成相应图像，支持多种风格和分辨率

【技术实现】
- 基础模型：Stable Diffusion v1.5 (runwayml/stable-diffusion-v1-5)
- 备选方案：SiliconFlow API (Kwai-Kolors/Kolors模型)
- 深度学习框架：PyTorch
- 扩散模型库：Diffusers
- 图像处理：PIL (Pillow)

【模型参数】
- 默认图像尺寸：512x512
- 推理步数：15-20步（可配置）
- 引导尺度：6.0-7.5（可配置）
- 设备支持：CUDA (GPU) / CPU
- 数据类型：float16 (GPU) / float32 (CPU)

【性能优化】
- 启用注意力切片 (enable_attention_slicing)
- 启用xFormers内存高效注意力 (enable_xformers_memory_efficient_attention)
- 确定性采样 (eta=0.0)
- 轻量级模型模式

【API配置】
- SiliconFlow API地址：https://api.siliconflow.cn/v1/images/generations
- 使用模型：Kwai-Kolors/Kolors
- 认证方式：Bearer Token

【主要方法】
- generate_image(): 生成图像
- image_to_base64(): 图像转Base64
- generate_image_base64(): 生成图像并返回Base64
- get_sd_model(): 获取模型实例（单例模式）

【错误处理】
- API调用失败时自动切换到本地模型
- 本地模型失败时返回占位图像
- 详细的日志记录

3. LLM模型 (llm_model.py)
================================================================================

【模型功能】
- 核心功能：文本生成和动态叙事生成
- 支持功能：
  - 通用文本生成
  - 动态叙事故事生成
  - 故事场景解析
  - 连贯故事整合
  - 图像提示词增强

【技术实现】
- 后端服务：阿里云百炼API
- 使用模型：qwen3-vl-plus
- 通信协议：HTTP REST API
- 数据格式：JSON

【模型参数】
- 最大生成Token数：1000-3000（根据任务调整）
- 生成温度：0.7
- Top-p采样：0.9
- 故事长度选项：short(3场景) / medium(4场景) / long(5场景)

【API配置】
- API地址：https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions
- 认证方式：Bearer Token
- API Key：从环境变量DASHSCOPE_API_KEY或默认值获取

【主要方法】
- generate_text(): 通用文本生成
- generate_story(): 生成动态叙事故事
- generate_coherent_story(): 生成连贯故事
- _parse_story_scenes(): 解析故事场景
- _generate_enhanced_image_prompt(): 生成增强图像提示词
- _get_default_story_scenes(): 获取默认故事场景
- get_llm_model(): 获取模型实例（单例模式）

【故事生成流程】
1. 根据关键词和风格构建提示词
2. 调用LLM生成故事内容
3. 解析故事场景
4. 为每个场景生成增强的图像提示词
5. 返回结构化故事数据

- adapt_content_to_culture(): 内容文化适配

- 风格特征
- 应用示例
- 提示词关键词

- 向量维度：TF-IDF自动计算
