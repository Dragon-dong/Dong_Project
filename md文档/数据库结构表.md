# 跨模态内容生成系统数据库结构表

## 数据库表结构概览

| 表名 | 字段名 | 数据类型 | 描述 | 约束 |
|------|--------|----------|------|------|
| **users** | id | int | 用户ID | 主键 |
| | username | string | 用户名 | 唯一 |
| | email | string | 邮箱 | 唯一 |
| | password_hash | string | 密码哈希值 | 非空 |
| | created_at | datetime | 创建时间 | 自动生成 |
| | updated_at | datetime | 更新时间 | 自动生成 |
| **generation_history** | id | int | 历史记录ID | 主键 |
| | user_id | int | 用户ID | 外键(users.id) |
| | task_type | string | 任务类型 | 非空 |
| | input_data | json | 输入数据 | 非空 |
| | output_data | json | 输出数据 | 非空 |
| | status | string | 状态 | 非空 |
| | error_message | string | 错误信息 | 可选 |
| | duration | float | 执行时长(秒) | 可选 |
| | created_at | datetime | 创建时间 | 自动生成 |
| **user_preferences** | id | int | 偏好设置ID | 主键 |
| | user_id | int | 用户ID | 外键(users.id) |
| | preference_key | string | 偏好键 | 非空 |
| | preference_value | json | 偏好值 | 非空 |
| | created_at | datetime | 创建时间 | 自动生成 |
| | updated_at | datetime | 更新时间 | 自动生成 |
| **system_config** | id | int | 配置ID | 主键 |
| | config_key | string | 配置键 | 唯一 |
| | config_value | json | 配置值 | 非空 |
| | created_at | datetime | 创建时间 | 自动生成 |
| | updated_at | datetime | 更新时间 | 自动生成 |
| **Redis缓存** | key | string | 缓存键 | 唯一 |
| | value | json | 缓存值 | 非空 |
| | expire | int | 过期时间(秒) | 可选 |

## 表关系说明

| 关系类型 | 源表 | 目标表 | 描述 |
|----------|------|--------|------|
| 一对多 | users | generation_history | 一个用户可以有多个生成历史记录 |
| 一对多 | users | user_preferences | 一个用户可以有多个偏好设置 |
| 无直接关系 | system_config | - | 系统配置表，独立存储系统级配置 |
| 无直接关系 | Redis缓存 | - | 用于提高系统性能的临时数据存储 |

## 数据库设计特点

| 特点 | 说明 | 优势 |
|------|------|------|
| JSON存储 | 使用JSON格式存储复杂数据 | 提高数据存储的灵活性，便于存储和查询复杂结构 |
| 外键关联 | 通过外键建立表之间的关系 | 保证数据一致性，防止数据冗余 |
| 时间戳 | 所有表都包含创建时间和更新时间字段 | 便于跟踪数据变更，支持审计和日志记录 |
| Redis缓存 | 用于缓存频繁访问的数据 | 提高系统响应速度，减少数据库负载 |
| 错误处理 | 生成历史表包含错误信息字段 | 便于系统监控和问题排查 |

## 数据流向

1. **用户管理**：用户注册/登录 → users表存储用户信息
2. **内容生成**：用户请求 → 模型处理 → generation_history表记录生成过程和结果
3. **用户偏好**：用户设置偏好 → user_preferences表存储偏好设置
4. **系统配置**：管理员设置配置 → system_config表存储系统配置
5. **缓存管理**：频繁访问的数据 → Redis缓存 → 提高系统性能

此数据库设计支持系统的用户管理、生成历史记录、用户偏好设置和系统配置等功能，为跨模态内容生成系统提供了可靠的数据存储和管理基础。
