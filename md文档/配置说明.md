# 跨模态内容生成系统 - 配置说明

## 一、环境要求

### 1. 操作系统
- Windows 10/11
- macOS 10.15+
- Linux (Ubuntu 18.04+)

### 2. Python版本
- Python 3.8+
- 推荐使用 Python 3.10+ 以获得最佳性能

### 3. 硬件要求
- **CPU**：至少 4 核处理器

- **存储空间**：至少 20GB 可用空间
- **网络**：稳定的网络连接（用于调用外部API）

## 三、数据库配置

### 1. MySQL配置

#### 安装MySQL
- **Windows**：下载并安装 [MySQL Installer](https://dev.mysql.com/downloads/installer/)
- **macOS**：使用 Homebrew 安装 `brew install mysql`
- **Linux**：使用包管理器安装 `sudo apt install mysql-server`

#### 配置MySQL

1. 启动MySQL服务
2. 登录MySQL
   ```bash
   mysql -u root -p
   ```
3. 创建数据库
   ```sql
   CREATE DATABASE cross_modal_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
   ```
4. 验证数据库创建成功
   ```sql
   SHOW DATABASES;
   ```

### 2. Redis配置（可选）

#### 安装Redis
- **Windows**：下载 [Redis for Windows](https://github.com/tporadowski/redis/releases)
- **macOS**：`brew install redis`
- **Linux**：`sudo apt install redis-server`

#### 启动Redis服务
- **Windows**：运行 `redis-server.exe`
- **macOS/Linux**：`redis-server`

## 四、环境变量配置

### 1. 创建 `.env` 文件

在项目根目录创建 `.env` 文件，配置以下环境变量：

```env
# 数据库配置
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=123456
MYSQL_DATABASE=cross_modal_system

# Redis配置
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=
REDIS_DB=0

# API配置
API_KEY=your_api_key_here
DEBUG=True

# 模型配置
MODEL_CACHE_DIR=./models
MAX_GENERATION_SIZE=1024
TIMEOUT=30

# 图片生成配置
IMAGE_WIDTH=512
IMAGE_HEIGHT=512
IMAGE_QUALITY=95

# 安全配置
SECRET_KEY=your_secret_key_here
ALLOWED_ORIGINS=*

# 日志配置
LOG_LEVEL=INFO
LOG_FILE=app.log
```

### 2. 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| MYSQL_HOST | MySQL主机地址 | localhost |
| MYSQL_PORT | MySQL端口 | 3306 |
| MYSQL_USER | MySQL用户名 | root |
| MYSQL_PASSWORD | MySQL密码 | 123456 |
| MYSQL_DATABASE | 数据库名称 | cross_modal_system |
| REDIS_HOST | Redis主机地址 | localhost |
| REDIS_PORT | Redis端口 | 6379 |
| API_KEY | 外部API密钥 | your_api_key_here |
| DEBUG | 是否开启调试模式 | True |
| MODEL_CACHE_DIR | 模型缓存目录 | ./models |
| IMAGE_WIDTH | 生成图像宽度 | 512 |
| IMAGE_HEIGHT | 生成图像高度 | 512 |

## 五、API密钥配置

### 1. SiliconFlow API（用于文生图）

1. 访问 [SiliconFlow](https://www.siliconflow.cn/)
2. 注册账号并获取API密钥
3. 将API密钥填入 `.env` 文件中的 `API_KEY` 字段

### 2. 阿里云百炼API（用于图生文）

1. 访问 [阿里云百炼](https://dashscope.aliyun.com/)
2. 注册账号并获取API密钥
3. 在 `models/llava_model.py` 中配置API密钥

## 九、常见问题与解决方案

### 1. 数据库连接失败

**问题**：`MySQL连接失败: 1045 (28000): Access denied for user 'root'@'localhost'`

**解决方案**：
- 检查 `.env` 文件中的数据库配置是否正确
- 确保MySQL服务正在运行
- 验证用户名和密码是否正确
- 确保用户有访问 `cross_modal_system` 数据库的权限

### 2. Redis连接失败

**问题**：`Redis连接失败: Error 10061 connecting to localhost:6379`

**解决方案**：
- 检查Redis服务是否正在运行
- 确保Redis配置正确
- Redis连接失败不会影响核心功能，系统会自动降级为无缓存模式

### 3. API调用失败

**问题**：`生成图像失败: API调用失败`

**解决方案**：
- 检查网络连接是否正常
- 验证API密钥是否正确
- 检查API服务是否可用
- 查看系统日志获取详细错误信息

### 4. 模型加载失败

**问题**：`模型加载失败，返回模拟数据`

**解决方案**：
- 确保网络连接正常
- 检查模型文件是否完整
- 系统会自动使用模拟数据，不影响基本功能

### 5. 端口被占用

**问题**：`Error: [Errno 48] Address already in use`

**解决方案**：
- 关闭占用8000端口的进程
- 修改 `main.py` 中的端口配置

## 十、性能优化配置

### 1. 模型参数优化

| 参数 | 建议值 | 说明 |
|------|--------|------|
| num_inference_steps | 15-20 | 推理步数，减少可提高速度 |
| guidance_scale | 7.0-7.5 | 引导尺度，影响生成质量 |
| temperature | 0.2-0.7 | 文本生成温度，影响多样性 |
| top_p | 0.9 | 采样阈值，影响生成质量 |

### 2. 服务器配置

- **多进程运行**：使用gunicorn启动多个进程
- **缓存优化**：启用Redis缓存，提高重复请求的响应速度
- **异步处理**：使用FastAPI的异步特性，提高并发处理能力

### 3. 网络优化

- 使用稳定的网络连接
- 考虑使用CDN加速静态资源
- 优化API调用频率，避免频繁请求
