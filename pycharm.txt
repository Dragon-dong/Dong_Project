# 在PyCharm中配置运行项目的步骤

## 1. 配置Python解释器

从截图中可以看到，当前缺少有效的Python解释器。按照以下步骤配置：

1. **选择解释器**：
   - 点击 `<No interpreter>` 下拉菜单
   - 选择 `Add Interpreter...`
   - 在弹出的窗口中，可以选择：
     - **Existing environment**：使用已有的Python环境
     - **New environment**：创建新的虚拟环境（推荐）
     - **Conda environment**：使用Conda环境

2. **创建虚拟环境**（推荐）：
   - 选择 `New environment`
   - 选择 `Virtualenv` 或 `Conda`
   - 选择基础Python解释器
   - 点击 `OK` 创建环境

## 2. 安装项目依赖

配置好解释器后，需要安装项目依赖：

1. **使用PyCharm的包管理器**：
   - 打开 `File` > `Settings` > `Project` > `Python Interpreter`
   - 点击 `+` 按钮
   - 输入 `requirements.txt` 中的依赖包名，逐个安装
   - 或者点击 `Install from requirements.txt` 按钮，选择项目中的 `requirements.txt` 文件

2. **使用终端安装**：
   - 在PyCharm的终端中运行：
     ```bash
     pip install -r requirements.txt
     ```

## 注意事项

1. **依赖安装可能需要时间**：特别是 `diffusers` 和 `transformers` 等大型库
2. **模型下载**：首次运行时，系统可能会下载一些模型文件
3. **端口占用**：如果8000端口被占用，可以修改 `main.py` 中的端口设置
4. **环境变量**：确保 `.env` 文件中的配置正确

如果遇到问题：

2. 检查依赖是否全部安装
3. 检查数据库连接是否正常
4. 查看PyCharm的运行日志，了解具体错误信息
